# MiniiERP

A REST API for managing a mini ERP system with modules for inventory, sales, and purchases.

## Overview

This project is a minimal ERP (Enterprise Resource Planning) system built with modern web technologies. It provides core functionality for managing inventory, sales, and purchase operations.

## Features

- Inventory management
- Sales tracking
- Purchase order processing
- User authentication and authorization

## Getting Started

1. Clone the repository
2. Install dependencies with `npm install`
3. Run the application with `npm start`
